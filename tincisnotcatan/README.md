# Home assistant add-on: tinc is not catan

An online version of Settlers of Catan with the option to play with Advanced Economic features. 

_Thanks to everyone having starred my repo! To star it click on the image below, then it will be on top right. Thanks!_

[![Stargazers repo roster for @jdeath/homeassistant-addons](https://reporoster.com/stars/jdeath/homeassistant-addons)](https://github.com/jdeath/homeassistant-addons/stargazers)

## About
This addon is based on the [docker image](https://github.com/seansegal/tincisnotcatan). With many changes to better handle websockets.


## Configuration

```
port : 8000 #port you want to run on.
```

Webui can be found at `<your-ip>:port`.
